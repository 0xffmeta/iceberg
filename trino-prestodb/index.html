Trino (formerly Presto SQL)¶

The Iceberg connector is available with PrestoSQL, which is rebranded as Trino since January 2021. This page is kept for backwards compatibility only. The documentation for Iceberg’s Trino connector has moved to the Trino website. Please consult the Iceberg section of the Trino docs for more information.

PrestoDB¶

The Presto trademark is now exclusively reserved for PrestoDB. The documentation for Iceberg’s PrestoDB connector is in the PrestoDB website. Please consult the Iceberg section of the PrestoDB docs for more information.
